# 🩺 Diabetes Risk Predictor Web App

## 🚀 Overview

A web-based application that predicts diabetes risk using a trained deep learning model.

## 🛠 Tech Stack

- Frontend: Streamlit
- Backend: Python
- ML Model: PyTorch
- Database: MongoDB Atlas
- Authentication: bcrypt

## 🔐 Features

- User Registration & Login
- Secure Password Hashing
- Diabetes Risk Prediction
- BMI Calculator
